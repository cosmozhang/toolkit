# SemEval 2014 Task on Broad-Coverage Semantic Dependency Parsing

This repository contains a basic software toolkit intended to be used in connection with the SemEval 2014 Task on Broad-Coverage Semantic Dependency Parsing. Detailed information about the task can be found at this website:

[http://alt.qcri.org/semeval2014/task8/](http://alt.qcri.org/semeval2014/task8/)

## Downloading

The project is currently only available via Git. We plan to make a download available when the software has stabilized.

## Building

After checking out the project from the repository, you should be able to build it using `ant'.

	% ant
